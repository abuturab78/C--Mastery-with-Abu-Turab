Lesson Details:

Pointers C++ mein variables ke memory addresses store karne ke liye use hote hain. Is lesson mein hum seekhenge:

Key Points:

  • Pointer Declaration: Pointer declare karne ke liye data type aur asterisk (*) ka use hota hai. 
                                
int* ptr; // Pointer to an integer
  • Pointer Initialization: Pointer ko kisi variable ke address se initialize kiya jata hai. 
                                
int num = 10;
int* ptr = # // Pointer initialized with address of num
  • Pointer Dereferencing: Pointer ke through value access karne ke liye asterisk (*) ka use hota hai. 
                                
int value = *ptr; // Dereferencing the pointer
  • Example: Ek program jo pointers ka use karta hai: 
                                
#include <iostream>
using namespace std;

int main() {
    int num = 10;
    int* ptr = #

    cout << "Value of num: " << num << endl;
    cout << "Address of num: " << &num << endl;
    cout << "Value of ptr: " << ptr << endl;
    cout << "Value pointed by ptr: " << *ptr << endl;

    return 0;
}

End of Lesson
