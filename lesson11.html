Lesson Details:

Structures C++ mein multiple data types ko ek single unit mein store karne ke liye use hote hain. Is lesson mein hum seekhenge:

Key Points:

  • Structure Declaration: Structure declare karne ke liye `struct` keyword ka use hota hai. 
                                
struct Student {
    int rollNo;
    string name;
    float marks;
};
  • Structure Initialization: Structure ke variables ko initialize kiya jata hai. 
                                
Student s1;
s1.rollNo = 1;
s1.name = "Abu Turab";
s1.marks = 95.5;
  • Structure Access: Structure ke members ko dot (.) operator ke through access kiya jata hai. 
                                
cout << "Roll No: " << s1.rollNo << endl;
cout << "Name: " << s1.name << endl;
cout << "Marks: " << s1.marks << endl;
  • Example: Ek program jo structure ka use karta hai: 
                                
#include <iostream>
#include <string>
using namespace std;

struct Student {
    int rollNo;
    string name;
    float marks;
};

int main() {
    Student s1;
    s1.rollNo = 1;
    s1.name = "Abu Turab";
    s1.marks = 95.5;

    cout << "Roll No: " << s1.rollNo << endl;
    cout << "Name: " << s1.name << endl;
    cout << "Marks: " << s1.marks << endl;

    return 0;
}

End of Lesson
