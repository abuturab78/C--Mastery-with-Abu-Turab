Lesson Details:

Operators C++ mein variables aur values par operations perform karne ke liye use hote hain. Is lesson mein hum seekhenge:

Key Points:

  • Arithmetic Operators: Addition (+), Subtraction (-), Multiplication (*), Division (/), Modulus (%). 
                                
int a = 10, b = 3;
int sum = a + b; // sum = 13
int difference = a - b; // difference = 7
int product = a * b; // product = 30
int quotient = a / b; // quotient = 3
int remainder = a % b; // remainder = 1
  • Relational Operators: Equal to (==), Not equal to (!=), Greater than (>), Less than (<), Greater than or equal to (>=), Less than or equal to (<=). 
                                
int x = 5, y = 10;
bool isEqual = (x == y); // isEqual = false
bool isGreater = (x > y); // isGreater = false
  • Logical Operators: AND (&&), OR (||), NOT (!). 
                                
bool a = true, b = false;
bool result1 = (a && b); // result1 = false
bool result2 = (a || b); // result2 = true
bool result3 = !a; // result3 = false
  • Example: Ek program jo operators ka use karta hai: 
                                
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;
    cout << "Sum: " << (a + b) << endl;
    cout << "Difference: " << (a - b) << endl;
    cout << "Product: " << (a * b) << endl;
    cout << "Quotient: " << (a / b) << endl;
    cout << "Remainder: " << (a % b) << endl;
    return 0;
}
