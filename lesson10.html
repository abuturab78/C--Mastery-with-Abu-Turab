Lesson Details:

Strings C++ mein text data store karne ke liye use hote hain. Is lesson mein hum seekhenge:

Key Points:

  • String Declaration: String declare karne ke liye `string` data type ka use hota hai. 
                                
string name = "Abu Turab"; // String declaration
  • String Operations: Strings par kuch common operations hote hain jaise concatenation, length, aur comparison. 
                                
string firstName = "Abu";
string lastName = "Turab";
string fullName = firstName + " " + lastName; // Concatenation
int length = fullName.length(); // Length of string
  • Example: Ek program jo strings ka use karta hai: 
                                
#include <iostream>
#include <string>
using namespace std;

int main() {
    string firstName = "Abu";
    string lastName = "Turab";
    string fullName = firstName + " " + lastName;

    cout << "Full Name: " << fullName << endl;
    cout << "Length of Full Name: " << fullName.length() << endl;

    return 0;
}
