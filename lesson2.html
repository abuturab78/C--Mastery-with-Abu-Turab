Lesson Details:

Variables aur data types C++ programming ke basic building blocks hain. Is lesson mein hum seekhenge:

Key Points:

  • Variables: Variables data store karne ke liye use hote hain. Har variable ka ek unique naam aur data type hota hai. 
                                
int age = 25; // 'int' data type ka use karke 'age' variable banaya gaya hai.
  • Data Types: C++ mein kuch common data types hain:
    • int: Integer values ke liye (e.g., 10, -5).
    • float: Decimal values ke liye (e.g., 3.14, -0.5).
    • double: Float se bhi precise decimal values ke liye (e.g., 3.14159).
    • char: Single character ke liye (e.g., 'A', 'b').
    • bool: Boolean values ke liye (true ya false).
  • Example: Ek program jo variables aur data types ka use karta hai: 
                                
#include <iostream>
using namespace std;

int main() {
    int age = 25;
    float height = 5.9;
    char grade = 'A';
    bool isStudent = true;

    cout << "Age: " << age << endl;
    cout << "Height: " << height << endl;
    cout << "Grade: " << grade << endl;
    cout << "Is Student: " << isStudent << endl;

    return 0;
}
