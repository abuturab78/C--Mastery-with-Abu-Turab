Lesson Details:

Arrays C++ mein multiple values store karne ke liye use hote hain. Is lesson mein hum seekhenge:

Key Points:

  • Array Declaration: Ek array declare karne ke liye data type, array ka naam, aur size specify karna hota hai. 
                                
int numbers[5]; // 5 integers ka array
  • Array Initialization: Array ko declare karte waqt values assign ki ja sakti hain. 
                                
int numbers[5] = {1, 2, 3, 4, 5};
  • Array Access: Array ke elements ko index ke through access kiya jata hai. 
                                
int firstElement = numbers[0]; // first element
int secondElement = numbers[1]; // second element
  • Example: Ek program jo array ka use karta hai: 
                                
#include <iostream>
using namespace std;

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};

    for (int i = 0; i < 5; i++) {
        cout << "Element " << i << ": " << numbers[i] << endl;
    }

    return 0;
}
